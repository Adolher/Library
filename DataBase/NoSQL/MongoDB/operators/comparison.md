# Comparison Operator

`db.<collection>.find( { field: { <operator>: <value> } } )`

| \<operator> | Description |
| :---: | :--- |
| $eq | equal |
| $gt | greater than |
| $gte | greater than or equal |
| $in | in \<array> |
| $lt | less than |
| $lte | less than or equal |
| $ne | not equal |
| $nin | not in \<array> |
